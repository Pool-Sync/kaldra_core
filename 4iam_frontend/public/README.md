## Asset Guidelines

When replacing placeholders with real assets:

1. **Logo**: Should follow KALDRA branding guidelines
2. **Icon**: Provide multiple sizes (16x16, 32x32, 64x64, 128x128, 256x256)
3. **Favicon**: Create .ico file with multiple sizes embedded
4. **Format**: Use SVG for scalable graphics, PNG for raster images
5. **Optimization**: Compress all images for web performance

## Integration

Assets are automatically available at `/logo.svg`, `/icon.png`, etc. in the application.
