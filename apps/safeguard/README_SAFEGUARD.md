# KALDRA-SAFEGUARD — Narrative & Bias Guard App (v0.1)

Aplicação mínima para monitoramento narrativo e de viés usando o KALDRA CORE.

- Entrada: textos monitorados (`raw_text` ou lista de textos).
- Pipeline principal: `generate_kaldra_signal`.
- Saída: JSON com campos de viés, regime TW e risco narrativo derivado.
